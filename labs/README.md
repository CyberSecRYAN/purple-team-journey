# Cybersecurity Labs

This folder contains hands-on cybersecurity labs focused on SOC analysis, blue-team fundamentals, networking, Linux, log analysis, and threat detection.

## Lab Focus Areas

* SOC alert triage
* Wireshark packet analysis
* Linux command line practice
* Log analysis
* Network troubleshooting
* MITRE ATT&CK mapping
* Incident response documentation
* Security tool practice

## Lab Format

Each lab will include:

* Objective
* Tools used
* Scenario
* Steps performed
* Findings
* Lessons learned
* Screenshots when available
